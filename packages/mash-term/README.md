# `mash-term`

# APIs

```
- focus/blur terminal
- type in string
  - Terminal#prompt
  - detect enter to emit the string
- show/hide cursor
- append lines
  - Terminal#writeln
- configure
  - Terminal#configure
    - prompt
    - background
- scroll
  - Terminal#scroll(rowToScroll: number)
- scroll to bottom
  - Terminal#scrollToBottom
```
